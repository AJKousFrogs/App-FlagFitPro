FlagFit Pro Component Library

This is the living documentation for all reusable components in FlagFit Pro. Each component includes HTML markup, CSS classes, usage examples, and accessibility guidelines.

Quick Start: Copy the HTML snippets from any component below and paste them into your pages. All components use the design system CSS classes from src/css/main.css.

Atoms

Basic building blocks that cannot be broken down further.

Button

Primary Button

All Sizes

All Variants

With Icons

Show HTML 
<button class="btn btn-primary btn-md">
  Start Training
</button>

<!-- With Icon -->
<button class="btn btn-primary btn-md">
  <i data-lucide="play" style="width: 18px; height: 18px;"></i>
  Start Training
</button>

<!-- Disabled State -->
<button class="btn btn-primary btn-md" disabled>
  Disabled
</button>

Variants

  • .btn-primary - Main actions (green gradient)
  • .btn-secondary - Supporting actions (outlined)
  • .btn-tertiary - Low emphasis (text with underline)

Sizes

  • .btn-xs - Extra small (28px height)
  • .btn-sm - Small (36px height)
  • .btn-md - Medium (44px height, default)
  • .btn-lg - Large (52px height)
  • .btn-xl - Extra large (60px height)

Accessibility

  • Focus indicator included via :focus-visible
  • Icon size must be explicitly set (16px, 18px, 20px)
  • Button text should be clear and action-oriented
  • Disabled state prevents interaction

Input

Basic Input

States

Show HTML 
<input type="text" class="form-input" placeholder="Enter text...">

<!-- Error State -->
<input type="text" class="form-input error" placeholder="Error" value="Invalid">

<!-- Success State -->
<input type="text" class="form-input success" placeholder="Success" value="Valid">

<!-- Disabled State -->
<input type="text" class="form-input" placeholder="Disabled" disabled>

States

  • .error - Validation error (red border)
  • .success - Valid input (green border)
  • disabled - Non-interactive state

Accessibility

  • Always pair with <label> element
  • Use 16px font size (prevents iOS zoom)
  • Include aria-describedby for error messages

Textarea

Basic Textarea

Show HTML 
<div class="form-group">
  <label for="textarea-id" class="form-label">Description</label>
  <textarea id="textarea-id" name="description" class="form-input form-textarea" rows="4"></textarea>
</div>

Usage

  • Multi-line text input for longer content
  • Use .form-textarea class
  • Set rows attribute for visible lines
  • Supports same states as input (error, success, disabled)

Select/Dropdown

Basic Select

Show HTML 
<div class="form-group">
  <label for="select-id" class="form-label">Position</label>
  <select id="select-id" name="position" class="form-input form-select">
    <option value="">Select position</option>

Usage

  • Dropdown selection from list of options
  • Use .form-select class
  • First option should be empty or placeholder
  • Supports multiple attribute for multi-select

Icon

Lucide Icons

Show HTML 
<!-- Lucide Icon -->
<i data-lucide="home" style="width: 24px; height: 24px;"></i>

<!-- With Color -->
<i data-lucide="football" style="width: 24px; height: 24px; color: var(--color-brand-primary);"></i>

Usage

  • Use Lucide icons via data-lucide attribute
  • Always set explicit width and height
  • Icons are initialized automatically via icon-helper.js
  • Common sizes: 16px, 18px, 20px, 24px

Form Input

Basic Form Input

We'll never share your email
Password must be at least 8 characters
Show HTML 
<div class="form-group">
  <label for="username" class="form-label">Username</label>
  <input type="text" id="username" name="username" class="form-input" placeholder="Enter your username">
</div>

<!-- With Helper Text -->
<div class="form-group">
  <label for="email" class="form-label">Email Address</label>
  <input type="email" id="email" name="email" class="form-input" placeholder="athlete@example.com">
  <small class="form-hint">We'll never share your email</small>
</div>

<!-- With Error -->
<div class="form-group">
  <label for="password" class="form-label required">Password</label>
  <input type="password" id="password" name="password" class="form-input error">
  <div class="form-error" role="alert">Password must be at least 8 characters</div>
</div>

Structure

  • .form-group - Container wrapper
  • .form-label - Label styling
  • .form-input - Input field
  • .form-hint - Helper text
  • .form-error - Error message

Accessibility

  • Always include <label> with matching id
  • Error messages use role="alert"
  • Required fields marked with .required class
