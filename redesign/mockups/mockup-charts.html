SHIP

How to build these in production

You already ship Chart.js v4.5. Keep it for time-series + bar charts (#03, #07, #12). For the rest, drop the dependency and render SVG directly from your component templates — they're maybe 60-120 lines each, infinitely themeable, and 10× lighter than any chart library.

CHART.JS v4
Time-series + bars
Charts #03, #05, #07, #12. Already wired into your `chart.config.ts`.
PURE SVG
Gauges + sparklines + radars
Charts #01, #02, #06, #11. No deps, perfectly themeable via CSS vars.
D3.js (NEW)
Heatmaps + beeswarm + field viz
Charts #04, #08, #09, #10. Add `d3` for the spatial/distribution patterns only.
