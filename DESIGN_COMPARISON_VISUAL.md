## Animation Details

### Hover Sequence (300ms total)
```
Frame 0ms:   Normal state
             Border: 4px left
             Position: x=0
             Shadow: 2px
             
Frame 100ms: Border animating
             Border: 6px → 8px
             
Frame 200ms: Card moving
             Position: x=0 → x=4px
             Shadow: 2px → 8px
             
Frame 300ms: Complete
             Border: 8px (fully expanded)
             Position: x=4px (slid right)
             Shadow: 8px (enhanced depth)
             Icon: rotated -5deg
```

## Technical Comparison

### Before
```css
.recommendation-card {
  border-left: 3px solid var(--color);
  background: rgba(color, 0.04);
  padding: var(--space-4);
  border-radius: var(--radius-md);
}
/* Simple, flat, basic */
```

### After
```scss
.recommendation-card {
  border-radius: 16px;
  border: 1px solid var(--color-border);
  padding: var(--space-5);
  background: linear-gradient(135deg, rgba(color, 0.08), rgba(color, 0.02));
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04);
  position: relative;
  
  &::before {
    content: "";
    position: absolute;
    left: 0;
    width: 4px;
    height: 100%;
    background: linear-gradient(180deg, color-500, color-600);
    transition: width 0.3s;
  }
  
  &:hover {
    transform: translateX(4px);
    box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08);
    
    &::before { width: 8px; }
    
    .rec-icon {
      transform: scale(1.1) rotate(-5deg);
    }
  }
}
/* Modern, dynamic, engaging */
```
