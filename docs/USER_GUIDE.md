# Wireframe App User Guide

## 🎯 **How to Navigate the Complete Wireframe App**

This guide will help you explore all the user flows and features in the comprehensive wireframe app.

## 🚀 **Getting Started**

### **1. Initial Access**
- Open the app at `http://localhost:4005`
- You'll see the **Login Page** with role auto-detection
- Click "🔐 Login with Email" to proceed (no actual authentication required)

### **2. Onboarding Experience**
- After login, the **Onboarding Modal** will appear automatically
- This simulates a new user's first-time setup experience
- Complete the 5-step process or click "Skip for Now"

## 📱 **Complete User Flow Walkthrough**

### **Flow 1: New User Journey**
1. **Login Page** → Click "Login with Email"
2. **Onboarding Modal** → Complete setup process
3. **Dashboard** → View personalized welcome and stats
4. **Navigation** → Explore all sections

### **Flow 2: Training Session**
1. **Dashboard** → Click "Start Training" button
2. **Training Page** → View AI coach message and focus areas
3. **Drill Selection** → Click "Start Drill" on any recommended drill
4. **Progress Tracking** → Monitor current streak and XP

### **Flow 3: Team Communication**
1. **Dashboard** → Click "Team Chat" button
2. **Community Page** → View team overview and chat rooms
3. **Chat Interaction** → Type in message input and click "Send"
4. **Chemistry Tracking** → Notice chemistry ratings on messages

### **Flow 4: Profile Management**
1. **Dashboard** → Click "Profile" in navigation
2. **Profile Page** → View personal information and stats
3. **Edit Features** → Click "[Edit]" buttons to see form interactions
4. **Physical Metrics** → Review universal rankings and percentiles

### **Flow 5: Tournament Planning**
1. **Dashboard** → Click "Tournaments" in navigation
2. **Tournaments Page** → View upcoming events and schedules
3. **Nutrition Planning** → Review dietary restrictions and preferences
4. **Break Optimization** → See structured 30-minute break protocols

- **Physical Profile Cards**: Height, weight, 40-yard dash with percentile rankings
- **Game Statistics**: Latest game performance with coach comments

- **Weekly Schedule**: Visual calendar with practice/game scheduling

- **AI Coach Messages**: Personalized training recommendations

### **Community Features**
- **Team Overview**: Active players, chemistry ratings, next game

### **Tournament Features**
