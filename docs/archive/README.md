# Archived Documentation

This folder contains historical documentation that has been archived to keep the main documentation folder clean and focused on current, active documentation.

## 📁 Structure

- **implementation-summaries/** - Historical implementation summaries and completion reports
- **wireframes/** - Individual wireframe documentation (main wireframe docs kept in parent)
- **reports/** - Historical audit reports and status reports
- **historical/** - Outdated or redundant documentation

## 📋 What's Here

These files document **completed work** and **historical context** but are not needed for **current development**. They are preserved for reference but moved out of the main documentation folder.

## 🔍 Finding Information

- **Current documentation:** Check the parent `docs/` folder
- **Historical context:** Check this archive folder
- **Component library:** See `component-library.html` in project root
- **Design system:** See `DESIGN_SYSTEM_DOCUMENTATION.md` in project root

## 📝 Note

If you need information from archived files, they're still here - just organized better. The main `docs/` folder now contains only active, current documentation.
