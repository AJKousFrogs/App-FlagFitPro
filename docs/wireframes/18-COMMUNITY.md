## Post Types

| Type | Description |
|------|-------------|
| Text | Plain text content |
| Image | Photo attachment |
| Video | Video attachment |
| Poll | Interactive poll with options |

## Post Actions

| Action | Description |
|--------|-------------|
| Like | Heart reaction |
| Comment | Add comment |
| Share | Share to others |
| Bookmark | Save post |
