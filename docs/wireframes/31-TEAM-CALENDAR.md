# Wireframe: Team Calendar (Player View)

**Route:** `/calendar`  
**Users:** Players (view events, RSVP)  
**Status:** ⚠️ Needs Implementation  
**Source:** Referenced in `FEATURE_DOCUMENTATION.md` §46

---

## Purpose

Allows players to view all team events, RSVP to practices/games/tournaments, indicate ride needs, and sync with their personal calendars.

## Event Types

| Type         | Icon | Examples                        |
| ------------ | ---- | ------------------------------- |
| Practice     | 🏋️   | Regular practice, skill session |
| Game         | 🏈   | League game, friendly           |
| Tournament   | 🏆   | Multi-game tournament           |
| Team Meeting | 📋   | Film session, planning          |
| Team Event   | 🎉   | Team dinner, bonding            |
| Travel       | ✈️   | Travel day                      |

## RSVP Responses

| Response | Color     | Meaning              |
| -------- | --------- | -------------------- |
| Going    | 🟢 Green  | Confirmed attendance |
| Can't Go | 🔴 Red    | Declined             |
| Maybe    | 🟡 Yellow | Tentative            |
| Pending  | ⚪ Gray   | Not yet responded    |
