- Full inventory management is a coach settings sub-feature
- Equipment checklist for practice is in `C06-PRACTICE-PLANNER.md`
- Could be a future standalone page if needed

### §15 Officials Management
- League/admin-level feature
- Out of scope for team-focused app
- Could integrate with Superadmin if needed

### §30-32 Professional Reports
- These are **export-focused** features, not standalone dashboards
- Nutritionist report = PDF export from wellness/nutrition data
- Physio report = PDF export from injury/recovery data
- Psychology report = PDF export from wellness/stress data
- All export functionality noted in relevant wireframes

### §44 Scouting Reports
- Implemented through Film Room tagging system
- Coach tags opponent tendencies in film
- Not a separate page; integrated into `C14-FILM-ROOM-COACH.md`
